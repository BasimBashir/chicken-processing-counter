Settings

Changes apply to the running stream on the next frame — counts are not reset.

Detection

0.25

Min YOLO score to accept single-legged / slaughtered detections. Lower = catch more birds (fewer misses) but more false positives; higher = stricter.

0.45

Min score for the empty_shackles class only, tuned separately from the chicken classes. Lower detects more empty hooks.

0.45

Overlap threshold for merging duplicate boxes (agnostic NMS). Lower = more aggressive merging (risk: a chicken next to a shackle gets suppressed); higher keeps adjacent boxes.

Inference resolution (multiple of 32). Higher = better small-object accuracy but slower. The model is trained at 1280.

Counting line

0.65

Where the vertical counting line sits across the frame (fraction of width). Move it to where chickens are clearly separated.

15

Half-width of the band around the line. Wider tolerates bbox flicker / frame stutter so fast birds aren't missed. 0 = single-pixel tripwire.

34

Belt travel per frame in pixels — seeds the per-track velocity estimator (which then self-tunes from motion). Set near your real belt speed (~34 at the 1280 sub-stream).

Overlay labels (no effect on the count)

90

Max px an on-screen #ID may jump between frames before it's treated as a new object. Affects the overlay labels only — not the count.

2

Frames an on-screen #ID survives without a match before being dropped. Overlay labels only — not the count.
